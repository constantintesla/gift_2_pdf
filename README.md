# gift_2_pdf

Конвертер вопросов в формате GIFT в аккуратный PDF.

## Возможности

- Поддержка категорий `$CATEGORY:`
- Типы вопросов: `multiple choice`, `true/false`, `matching`, `numerical`, `short answer`, `description`
- Авто-подбор кодировки входного файла
- Кириллица в PDF (через системные шрифты Windows)

## Требования

- Python 3.8+
- Windows (для системных шрифтов Arial/Times New Roman)

Зависимости:

- `reportlab` (если не установлен, скрипт попробует установить его сам)

## Установка

```bash
python -m pip install reportlab
```

## Использование

```bash
python gift_to_pdf.py путь\к\файлу.gift
```

Необязательный второй аргумент — путь к выходному PDF:

```bash
python gift_to_pdf.py путь\к\файлу.gift путь\к\результату.pdf
```

Если аргументы не заданы, используется путь по умолчанию, заданный в `gift_to_pdf.py`.

## Пример GIFT

```gift
$CATEGORY: Математика
::Сложение:: Сколько будет 2+2? { =4 ~3 ~5 }

$CATEGORY: История
Столица Франции? { =Париж ~Лион ~Марсель }

::Истина:: Небо голубое. { T }

::Сопоставление:: Соотнесите
{
=Москва -> Россия
=Берлин -> Германия
}
```

## Примечания

- Комментарии `//` и `/* ... */` в GIFT игнорируются.
- HTML-разметка `[html]` поддерживается частично (b, i, br, p).
- В PDF выводятся правильные ответы (подходит для проверочных материалов).
